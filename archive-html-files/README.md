# Archived HTML Files

This folder contains the original HTML protocol files that were used in the initial development phase.

## Contents
- 344 HTML files from the original PDF extraction
- Combined protocol files (red.html, green.html, yellow.html)
- Individual numbered protocol files from all categories
- contents.html (original table of contents)

## Why Archived?
These files have been replaced by the markdown-based parser that uses [meems2025.md](../meems2025.md) as the source of truth. The markdown parser provides cleaner, more consistent data extraction and successfully parsed 101 protocols.

## Date Archived
February 10, 2026

## Note
These files are kept for reference purposes only. The PWA now uses data generated from meems2025.md via the `npm run parse-md` script.
